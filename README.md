## Nextjs GYM APP

## Built With
* Nextjs
* Typescript
* Tailwind CSS

### Live link
Check out the Website by visiting this [Link](https://nextjs-task-wine.vercel.app/).

## Install 
    using SSH:
    $ git clone git@github.com:knrbokhari/nextjs_task.git
    or using HTTPS:
    $ git clone https://github.com/knrbokhari/nextjs_task.git
    $ cd nextjs_task
    $ npm install

### Run the next dev
    $ npm run dev
### Run the build 
    $ npm run build
### Run start 
    $ npm start
